# Traffic Vehicle Detection Using YOLO

## Overview

An AI-powered traffic vehicle detection system built using YOLOv8, OpenCV, and Python. The project detects vehicles from traffic images and videos and supports custom dataset training.

## Features

- Vehicle Detection
- Frame Extraction from Videos
- Automatic Dataset Labeling
- Custom YOLO Model Training
- Weather Simulation Testing
- Traffic Image Processing

## Technologies

- Python
- YOLOv8
- OpenCV
- Ultralytics
- NumPy
- Roboflow
